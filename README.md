Stopwatch Web App 

.Overview

This project is a Stopwatch Web Application created as Task 2 for my internship.
It allows users to start, pause, reset, and track time accurately using JavaScript.
The UI is simple, clean, and responsive.

.Features

Start, Pause, and Reset functionality

Accurate time tracking (minutes, seconds, milliseconds)

Responsive design

Minimal and clean UI

Smooth button interaction

Technologies Used

HTML5

CSS3

JavaScript

.Folder Structure
/stopwatch-project
│── index.html
│── style.css
│── script.js

. How to Run

Download or clone the repository

Open index.html in your browser

Stopwatch will start working — no extra setup needed
